## Atelier 1 - backend + frontend + base de donnée :
On a construit une architecture complete avec frontend, backend et base de donnees relies via Docker Compose.

## Atelier 2 - Methodologie BDD
On a compris que les scenarios BDD permettent de definir le comportement attendu avant l'implementation technique.

## Atelier 3 - DC avec Vercel et Render
On a deploye le projet sur des plateformes cloud et validé la separation frontend/backend en production.

## Atelier 4 - automatisations des test unitaires
On a automatise l'execution des tests unitaires pour verifier rapidement les regressions sur Python et JavaScript.

## Atelier 5 - Ci, migration continue 
On a mis en place la CI avec GitHub Actions pour lancer automatiquement les tests a chaque push et pull request.

## Atelier 6 - Monitoring (graphana, ect...)
On a compris l'interet du monitoring pour surveiller la sante de l'application et detecter les problemes plus vite.
